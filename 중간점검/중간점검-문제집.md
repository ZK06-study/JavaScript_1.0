## 📚 문제 목록

### **문제 1** - 기본 문법 (변수 선언과 호이스팅)
**다음 코드의 실행 결과는 무엇인가요?**

```javascript
console.log(x);
var x = 5;
console.log(x);
```

**선택지:**
1. undefined, 5
2. ReferenceError, 5
3. 5, 5
4. undefined, undefined

**정답**: 1번
**해설**: var로 선언된 변수는 호이스팅되어 선언부가 상단으로 올라가지만, 초기화는 원래 위치에서 이루어집니다.

---

### **문제 2** - 데이터 타입 (typeof 연산자)
**다음 중 typeof 연산자의 결과가 다른 것은?**

**선택지:**
1. typeof null
2. typeof undefined
3. typeof {}
4. typeof []

**정답**: 2번
**해설**: typeof null은 "object"를 반환하는 JavaScript의 유명한 버그입니다. typeof undefined는 "undefined"를 반환합니다.

---

### **문제 3** - 함수 (함수 스코프와 클로저)
**다음 코드의 결과는 무엇인가요?**

```javascript
function outer() {
    let x = 10;
    function inner() {
        console.log(x);
    }
    return inner;
}
const fn = outer();
fn();
```

**선택지:**
1. 10
2. undefined
3. ReferenceError
4. null

**정답**: 1번
**해설**: 내부 함수 inner는 외부 함수 outer의 변수 x에 접근할 수 있습니다. 이것이 클로저의 핵심 개념입니다.

---

### **문제 4** - 객체 (this 바인딩)
**다음 코드에서 this.name의 값은?**

```javascript
const obj = {
    name: 'JavaScript',
    getName: function() {
        return this.name;
    }
};
const fn = obj.getName;
console.log(fn());
```

**선택지:**
1. JavaScript
2. undefined
3. null
4. ReferenceError

**정답**: 2번
**해설**: obj.getName을 변수에 할당하면, 함수는 객체와의 연결을 잃게 됩니다. 따라서 this는 전역 객체를 가리키게 됩니다.

---

### **문제 5** - 배열 (배열 메소드)
**다음 코드의 결과는?**

```javascript
const arr = [1, 2, 3, 4, 5];
const result = arr.filter(x => x > 2).map(x => x * 2);
console.log(result);
```

**선택지:**
1. [2, 4, 6, 8, 10]
2. [6, 8, 10]
3. [3, 4, 5]
4. [1, 2, 3, 4, 5]

**정답**: 2번
**해설**: filter(x => x > 2)는 [3, 4, 5]를 반환하고, map(x => x * 2)는 각 요소에 2를 곱해 [6, 8, 10]을 반환합니다.

---

### **문제 6** - 참조와 값 (객체 참조)
**다음 코드 실행 후 obj1.a의 값은?**

```javascript
const obj1 = { a: 1 };
const obj2 = obj1;
obj2.a = 2;
console.log(obj1.a);
```

**선택지:**
1. 1
2. 2
3. undefined
4. ReferenceError

**정답**: 2번
**해설**: obj1과 obj2는 같은 객체를 참조합니다. obj2.a를 변경하면 obj1.a도 함께 변경됩니다.

---

### **문제 7** - 문자열 (템플릿 리터럴) [코드 작성]
**다음 코드를 템플릿 리터럴로 작성하세요.**

```javascript
const name = 'JavaScript';
const version = 'ES2015';
const message = '안녕하세요, ' + name + ' ' + version + '입니다!';
```

**정답**: 
```javascript
const message = `안녕하세요, ${name} ${version}입니다!`;
```

**해설**: 템플릿 리터럴은 백틱(`)과 ${} 구문을 사용하여 문자열 보간을 수행합니다.

### **문제 8** - 함수 (화살표 함수) [코드 작성]
**다음 일반 함수를 화살표 함수로 변환하세요.**

```javascript
function multiply(a, b) {
    return a * b;
}
```

**정답**: 
```javascript
const multiply = (a, b) => a * b;
```

**해설**: 화살표 함수는 function 키워드를 생략하고 => 구문을 사용합니다. 단일 표현식인 경우 return을 생략할 수 있습니다.

### **문제 9** - 배열 (배열 구조분해) [코드 작성]
**다음 배열에서 구조분해 할당을 사용하여 첫 번째와 세 번째 요소를 추출하세요.**

```javascript
const colors = ['red', 'green', 'blue', 'yellow'];
// 첫 번째와 세 번째 요소를 first, third 변수에 할당
```

**정답**: 
```javascript
const [first, , third] = colors;
```

**해설**: 구조분해 할당에서 쉼표를 사용하여 건너뛸 요소를 표시할 수 있습니다.

### **문제 10** - 객체 (프로토타입과 상속)
**다음 코드의 결과는?**

```javascript
function Animal(name) {
    this.name = name;
}
Animal.prototype.speak = function() {
    return this.name + ' makes a sound';
};

function Dog(name) {
    Animal.call(this, name);
}
Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

const dog = new Dog('Buddy');
console.log(dog.speak());
```

**선택지:**
1. Buddy makes a sound
2. undefined makes a sound
3. TypeError
4. ReferenceError

**정답**: 1번
**해설**: Dog는 Animal을 상속받아 speak 메소드를 사용할 수 있습니다. call을 통해 Animal의 constructor를 실행합니다.

### **문제 11** - 고급 개념 (엄격 모드)
**엄격 모드에서 다음 코드의 결과는?**

```javascript
'use strict';
function test() {
    undeclaredVar = 10;
}
test();
```

**선택지:**
1. 10
2. undefined
3. ReferenceError
4. null

**정답**: 3번
**해설**: 엄격 모드에서는 선언되지 않은 변수에 값을 할당하면 ReferenceError가 발생합니다.

### **문제 12** - 함수 (즉시 실행 함수) [코드 작성]
**다음 즉시 실행 함수(IIFE)를 작성하여 변수 x를 10으로 초기화하고 반환하세요.**

**정답**: 
```javascript
(function() { 
    const x = 10; 
    return x; 
})();
```

**해설**: 즉시 실행 함수는 함수를 정의하자마자 실행하는 패턴입니다. 전역 스코프를 오염시키지 않고 변수를 캡슐화할 수 있습니다.

---

### **문제 13** - 배열 (reduce 메소드) [코드 작성]
**다음 배열의 모든 요소의 합을 구하는 reduce 코드를 작성하세요.**

```javascript
const numbers = [1, 2, 3, 4, 5];
// reduce를 사용하여 합계 구하기
```

**정답**: 
```javascript
const sum = numbers.reduce((acc, cur) => acc + cur, 0);
```

**해설**: reduce는 배열의 모든 요소를 하나의 값으로 축소할 때 사용합니다. 첫 번째 인자는 누적값, 두 번째 인자는 현재값입니다.

---

### **문제 14** - 객체 (Object.keys와 Object.values)
**다음 코드의 결과는?**

```javascript
const obj = { a: 1, b: 2, c: 3 };
console.log(Object.keys(obj).length);
console.log(Object.values(obj).reduce((a, b) => a + b));
```

**선택지:**
1. 3, 6
2. 3, 3
3. 6, 6
4. undefined, 6

**정답**: 1번
**해설**: Object.keys()는 객체의 키들을 배열로 반환하므로 길이는 3입니다. Object.values()는 [1, 2, 3]을 반환하고, 이를 reduce로 합하면 6입니다.

### **문제 15** - 고급 개념 (이벤트 루프 이해)
**다음 코드의 출력 순서는?**

```javascript
console.log('1');
setTimeout(() => console.log('2'), 0);
console.log('3');
```

**선택지:**
1. 1, 2, 3
2. 1, 3, 2
3. 2, 1, 3
4. 3, 1, 2

**정답**: 2번
**해설**: setTimeout은 비동기적으로 실행되므로, 동기 코드가 먼저 실행됩니다. 따라서 순서는 1, 3, 2입니다.
