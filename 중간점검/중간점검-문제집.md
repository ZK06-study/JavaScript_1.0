배열에서 중복된 요소를 제거하는 함수를 작성하세요.

**문제 설명:**
주어진 배열에서 중복된 요소를 제거하고 순서를 유지한 새 배열을 반환하는 함수를 구현하세요.

**함수 시그니처:**
```javascript

    // 여기에 코드를 작성하세요
}
```

**테스트 케이스:**

**함수 시그니처:**
```javascript

    // 여기에 코드를 작성하세요
    // tasks: Promise를 반환하는 함수들의 배열
```
