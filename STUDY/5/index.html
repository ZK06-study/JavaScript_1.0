템플릿 리터럴 Before & After

Before (기존 방식)

const name = '김철수';
const age = 25;
const message = '안녕하세요. 저는 ' + name + 
               '이고, ' + age + '세입니다.';

After (템플릿 리터럴)

const name = '김철수';
const age = 25;
const message = `안녕하세요. 저는 ${name}이고, ${age}세입니다.`;

문자열 속성과 메소드

기본 속성

  • length - 문자열 길이

검색 메소드

  • indexOf() - 위치 찾기
  • includes() - 포함 여부
  • startsWith() - 시작 확인
  • endsWith() - 끝 확인

추출 메소드

  • slice() - 부분 추출
  • substring() - 부분 추출
  • charAt() - 특정 문자

변환 메소드

  • toUpperCase() - 대문자
  • toLowerCase() - 소문자
  • replace() - 치환
  • trim() - 공백 제거

검색 메소드 활용

실습 문제

문제 1: 템플릿 리터럴 활용

이름과 나이를 변수로 받아서 자기소개 문장을 만드세요.

const name = '김철수';
const age = 20;
// 결과: "안녕하세요! 저는 김철수이고, 20세입니다."

💡 정답:

const message = `안녕하세요! 저는 ${name}이고, ${age}세입니다.`;

문제 2: 문자열 메소드 활용

이메일 주소에서 사용자명과 도메인을 분리하세요.

const email = 'user@example.com';
// 사용자명: 'user'
// 도메인: 'example.com'

💡 정답:

const atIndex = email.indexOf('@');
const username = email.slice(0, atIndex);
const domain = email.slice(atIndex + 1);

문제 3: 문자열 변환

CSV 문자열을 배열로 변환하고, 각 항목의 공백을 제거하세요.

const csv = ' apple , banana , orange ';
// 결과: ['apple', 'banana', 'orange']

💡 정답:

const result = csv.split(',').map(item => item.trim());
