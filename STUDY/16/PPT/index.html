#래퍼객체 #전역객체 #String #Math #Date

자바스크립트 객체의 3가지 분류

사용자 정의 객체

개발자가 필요에 의해 생성한 객체

const user = { name: 'Kim' };

내장 객체 (네이티브)

JS 엔진에 내장된 객체 (e.g., String, Number, Date)

const now = new Date();

호스트 객체

실행 환경이 제공하는 객체 (e.g., window, document)

window.alert('Hello');

오늘은 자바스크립트 언어 자체에 포함된 내장 객체에 집중합니다.

원시 값을 감싸는 마법, 래퍼 객체

원시 값(string, number, boolean)이 속성/메서드를 가질 수 있는 이유입니다.

const str = 'hello'; // 원시 값

// str.length에 접근하는 순간!
// 1. 임시 String 래퍼 객체 생성: new String('hello')
// 2. .length 속성 사용
// 3. 임시 객체 소멸

console.log(str.length); // 5
console.log(str.toUpperCase()); // "HELLO"

덕분에 우리는 원시 값을 객체처럼 편리하게 다룰 수 있습니다.

글로벌 파워, 전역 객체

모든 객체의 최상위, 전역 변수와 함수를 담는 보이지 않는 컨테이너입니다.

브라우저 환경

window.isNaN(123); // false
// 전역 객체가 window

Node.js 환경

global.setTimeout(() => {}, 1000);
// 전역 객체가 global

parseInt(), isNaN(), undefined, Infinity 등은 모두 전역 객체의 속성입니다.

인터랙티브: 문자열 분석기

분석 결과가 여기에 표시됩니다.

숫자의 모든 것, Number & Math

Number는 숫자 처리, Math는 수학적 계산을 위한 기능들을 제공합니다.

Number 객체

const num = 123.456;
num.toFixed(2); // "123.46" (문자열 반환)

Number.isInteger(123); // true
Number.parseInt('100px'); // 100

Math 객체

Math.max(1, 5, 2);    // 5
Math.random();        // 0과 1 사이의 난수
Math.round(1.6);      // 2 (반올림)
Math.floor(1.6);      // 1 (내림)

시간 여행자, Date 객체

날짜와 시간을 생성하고, 원하는 형식으로 가공할 수 있습니다.

const now = new Date(); // 현재 시간

console.log(now.getFullYear()); // 연도 (e.g., 2023)
console.log(now.getMonth() + 1);  // 월 (0부터 시작하므로 +1)
console.log(now.getDate());     // 일
console.log(now.getDay());      // 요일 (일요일=0, 월요일=1, ...)

// 사용자의 지역에 맞는 날짜/시간 형식
console.log(now.toLocaleString());

Date 객체를 활용하여 목표 날짜까지 남은 날을 계산해봅시다.

날짜를 선택해주세요.

패턴의 마법사, RegExp 객체

정규표현식은 문자열에서 특정 패턴을 검색하고 치환하는 강력한 도구입니다.

const str = 'My email is user@example.com';
const emailRegex = /[\w.-]+@[\w.-]+\.\w{2,4}/;

// 정규식 테스트
emailRegex.test(str); // true

// 패턴과 일치하는 문자열 찾기
str.match(emailRegex); 
// ["user@example.com", index: 12, ...]

// 패턴을 다른 문자열로 교체
str.replace(emailRegex, '******@*****.***');
// "My email is ******@*****.***"

핵심 정리

  • 래퍼 객체: 원시 값을 객체처럼 다룰 수 있게 해주는 임시 도우미
  • 전역 객체: `isNaN`, `parseInt` 등을 포함하는 최상위 객체
  • String: 문자열 조작을 위한 필수 메서드 모음
  • Number & Math: 숫자 처리와 수학 계산의 핵심
  • Date: 날짜와 시간 관리를 위한 표준 객체
  • RegExp: 복잡한 문자열 패턴 검색 및 처리의 전문가

내장 객체를 잘 활용하여 더 효율적이고 우아한 코드를 작성하세요!
