# JavaScript 학습 가이드 - 웹 PPT

JavaScript 기초부터 고급까지 체계적으로 학습할 수 있는 인터랙티브 웹 프레젠테이션입니다.

## 🌟 주요 기능

### 📚 학습 기능
- **24개 챕터** - JavaScript 기초부터 고급까지 체계적 학습
- **인터랙티브 코드 실행** - JavaScript 예제를 브라우저에서 바로 실행
- **프로그레스 트래킹** - 학습 진행 상황을 시각적으로 확인
- **검색 기능** - 전체 콘텐츠에서 원하는 내용 찾기

### 🎨 UI/UX 기능
- **5가지 테마** - Light, Dark, Blue, Green, Purple
- **반응형 디자인** - 데스크톱, 태블릿, 모바일 지원
- **키보드 단축키** - 효율적인 슬라이드 네비게이션
- **전체화면 모드** - 프레젠테이션에 최적화된 뷰

### ♿ 접근성
- **스크린 리더 지원** - 시각 장애인을 위한 접근성
- **키보드 네비게이션** - 마우스 없이도 완전한 조작 가능
- **고대비 모드** - 시각적 접근성 향상
- **모션 감소 지원** - 전정기관 장애 고려

## 🚀 시작하기

### 필요 조건
- 모던 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- 로컬 웹 서버 (선택사항, 권장)

### 설치 및 실행

1. **저장소 복제**
   ```bash
   git clone [repository-url]
   cd JS/web-ppt
   ```

2. **웹 서버 실행** (권장)
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (npx)
   npx serve .
   
   # Live Server (VS Code Extension)
   # 또는 다른 로컬 웹 서버 사용
   ```

3. **브라우저에서 열기**
   ```
   http://localhost:8000
   ```

### 직접 파일 열기
웹 서버 없이도 사용 가능하지만, CORS 정책으로 인해 일부 기능이 제한될 수 있습니다.

## 🎯 사용 방법

### 기본 네비게이션
- **다음 슬라이드**: `스페이스바`, `→`, `Page Down`
- **이전 슬라이드**: `←`, `Page Up`
- **첫 슬라이드**: `Home`
- **마지막 슬라이드**: `End`
- **특정 슬라이드 이동**: `Ctrl + G`

### 기능 단축키
- **전체화면**: `F` 또는 `F11`
- **전체화면 종료**: `Esc`
- **검색**: `Ctrl + F`
- **도움말**: `F1` 또는 `Shift + ?`
- **테마 변경**: 헤더의 테마 버튼 클릭

### 코드 실행
1. 슬라이드의 JavaScript 코드 블록에서 **"실행"** 버튼 클릭
2. 코드 편집기가 열리면 코드를 수정 가능
3. `Ctrl + Enter`로 코드 실행
4. 실행 결과를 즉시 확인

### 모바일 사용
- **스와이프**: 좌우 스와이프로 슬라이드 이동
- **터치**: 네비게이션 버튼 터치
- **Chapter 스크롤**: 상단 챕터 리스트를 좌우로 스크롤

8. **함수** - 함수 정의와 호출, 스코프

9. **제어 구문** - 조건문과 반복문

11. **배열** - 배열 조작과 메소드

15. **연산자 심화** - 고급 연산자들
16. **내장 객체** - Date, Math, RegExp 등

17. **함수형 프로그래밍** - FP 패러다임
18. **반복** - Iterator와 Generator

## 🔧 커스터마이징

### 테마 추가
`css/themes.css` 파일에 새 테마를 추가할 수 있습니다:

```css
[data-theme="custom"] {
    --primary-color: #your-color;
    --bg-color: #your-bg;
    /* 기타 CSS 변수들 */
}
```

### 슬라이드 콘텐츠 수정
- `docs/` 폴더의 마크다운 파일을 수정
- 또는 `data/slides.json`에서 직접 편집

### 새 기능 추가
각 모듈(`js/` 폴더)은 독립적으로 확장 가능합니다:
- `parser.js` - 콘텐츠 파싱 로직
- `navigation.js` - 네비게이션 시스템
- `codeRunner.js` - 코드 실행 엔진
- `app.js` - 메인 애플리케이션 로직
